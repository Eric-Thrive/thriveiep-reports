Measuring the Permission Gate: A Stress-Test Evaluation of Claude Code’s Auto Mode arXiv arXiv 2604.04978, Apr 4, 2026

Ji, Li, Jiang, Gao. First independent evaluation of Claude Code’s two-stage transcript-classifier permission gate. AmPermBench: 128 prompts across four DevOps task families. End-to-end FN rate 81.0% (95% CI 73.8-87.4) under stress-test vs Anthropic’s 17%; 36.8% of state-changing actions bypass the classifier scope entirely via in-project file edits (Tier 2).

We run Auto Mode and SYSTEM_ADMIN admin sessions. This is a direct argument for keeping the CLAUDE.md "Executing actions with care" rules loud, keeping git + Aptible backup discipline, and for continuing to gate destructive operations behind human confirmation even in auto mode.

PlayCoder: Making LLM-Generated GUI Code Playable arXiv arXiv 2604.19742, Apr 21 (Tencent)

PlayEval: 43 multilingual GUI apps (Python/TS/JS) scored on Play@k (end-to-end playability). 10 SOTA code LLMs score near-zero Play@3 despite high compile rates. PlayCoder’s multi-agent closed-loop repair reaches 38.1% Exec@3 and 20.3% Play@3.

The "compiles fine, still broken" gap is exactly the failure mode we hit in la3 narrative generation and bloom report drafts. Closed-loop repair with self-verification resonates with our pre-publish verify gate.

Precise Debugging Benchmark: Is Your Model Debugging or Regenerating? arXiv arXiv 2604.17338, Apr 19

Zhu, Chai, Wang, Liu, Bian. PDB converts existing coding datasets into debugging benchmarks with edit-level precision and bug-level recall. Frontier LLMs tend to regenerate over-edited solutions rather than patch surgically, a distinct failure mode from generation.

Reinforces our CLAUDE.md "Don’t add error handling, fallbacks, or validation for scenarios that can’t happen" and the bug-fix scope discipline.

Mistral OCR: structured-output document understanding API Mistral AI Current

API parses PDFs/images into interleaved ordered text + images, with tables, equations, and media as first-class elements. Supports "documents as prompts" with JSON-shaped output.

Worth a head-to-head against the current GLM-OCR pipeline on a small slice of the benchmark harness. Mistral OCR’s structured output could simplify the post-OCR PII pipeline.

[23]
TableSeq: Unified Generation of Structure, Content, and Layout arXiv arXiv 2604.16070, Apr 17

Hamdi, Tamasna, Boisson, Paquet. Image-only, end-to-end table structure + content + cell-localization in one decoder (interleaved HTML tags, cell text, discretized coordinate tokens). 95.23 TEDS / 96.83 S-TEDS on PubTabNet; 97.45 / 98.69 on FinTabNet.

Neuropsych reports the redactor ingests are table-heavy (score tables, subtest matrices). A single-decoder approach that skips external OCR could cut latency and error propagation in the PI Redactor pipeline.
